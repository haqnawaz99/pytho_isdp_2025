# Lesson 2: Division, Remainder, and Powers

## Duration

45 minutes

## Learning objective

Students will be able to use `//`, `%`, and `**` and explain their meanings.

## Key vocabulary

- floor division
- remainder
- power

## Starter

Ask students:

- If 17 mangoes are shared among 4 friends, how many does each friend get?
- How many are left over?

## Concept explanation

Python has special operators for equal share, leftover items, and powers.

- `//` gives the whole-number share
- `%` gives the remainder
- `**` gives the power

## Worked example 1

```python
print(17 // 4)
print(17 % 4)
```

## Worked example 2

```python
qurans = 25
shelves = 3

print(qurans // shelves)
print(qurans % shelves)
```

## Worked example 3

```python
print(2 ** 3)
print(5 ** 2)
```

## Guided practice

Students predict:

```python
print(20 // 6)
print(20 % 6)
print(3 ** 2)
```

## Independent practice

### Beginner

Find:

- `15 // 2`
- `15 % 2`

### Intermediate

Distribute 29 notebooks among 5 students and find share and leftover.

### Challenge

Write a program that prints:

- equal share
- leftover

for 41 candies shared among 6 children.

## Common mistakes

- thinking `%` means percentage
- using `/` when `//` is needed
- confusing multiplication `*` with power `**`

## Assessment checkpoint

1. What does `//` do?
2. What does `%` do?
3. What is `4 ** 2`?

## Exit ticket

Students write one example with `%`.

## Homework

Write a short program for equal distribution of books or pens using `//` and `%`.
