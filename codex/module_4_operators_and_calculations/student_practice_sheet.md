# Student Practice Sheet: Module 4

## Instructions

Solve each task in Python. First predict the answer, then run the code.

## Section 1: Arithmetic

1. Find:
   - `12 + 8`
   - `25 - 9`
   - `7 * 6`
   - `18 / 3`

2. Write a program for:
   - one pen price
   - number of pens
   - total cost

## Section 2: Division and Remainder

1. Find:
   - `20 // 3`
   - `20 % 3`
   - `4 ** 3`

2. Share 31 mangoes among 4 children:
   - equal share
   - leftover

## Section 3: Real-Life Calculations

1. A student buys 3 books at 200 rupees each. Find total cost.
2. A class revises 15 pages daily for 5 days. Find total pages.
3. A budget is 3000 rupees. If 950 rupees are spent, find amount left.

## Section 4: Comparison Operators

Write the result of:

1. `10 > 6`
2. `8 == 9`
3. `7 != 5`
4. `12 <= 10`

## Section 5: Logical Operators

Find the result of:

1. `(5 > 2) and (8 == 8)`
2. `(4 > 9) or (10 < 12)`
3. `not (6 == 6)`

## Section 6: Precedence

Find:

1. `5 + 2 * 3`
2. `(5 + 2) * 3`
3. `20 - 8 / 2`

## Challenge Section

1. Write a short budget program using at least three operators.
2. Create one comparison with `and`.
3. Create one expression where brackets change the answer.

## Teacher checking guide

Check whether students can:

- use arithmetic operators correctly
- explain remainder
- compare values accurately
- read boolean results
- use brackets deliberately
