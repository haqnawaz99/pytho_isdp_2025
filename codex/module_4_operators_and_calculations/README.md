# Module 4: Operators and Calculations

This module introduces arithmetic, comparison, and logical operators, along with simple calculation practice and operator precedence.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_arithmetic_operators.md`: `+`, `-`, `*`, `/`
- `lesson_2_division_remainder_and_powers.md`: `//`, `%`, `**`
- `lesson_3_real_life_calculations.md`: practical calculation programs
- `lesson_4_comparison_operators.md`: `==`, `!=`, `>`, `<`, `>=`, `<=`
- `lesson_5_logical_operators.md`: `and`, `or`, `not`
- `lesson_6_operator_precedence.md`: order of operations with brackets
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Teach arithmetic first with real-life contexts.
2. Keep comparison operators separate from `if` statements for now.
3. Use many prediction tasks before running code.
4. Reinforce that brackets improve clarity.
