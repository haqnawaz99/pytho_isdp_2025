# Lesson 3: Real-Life Calculations

## Duration

45 minutes

## Learning objective

Students will be able to write small programs using arithmetic operators in familiar situations.

## Key vocabulary

- total
- cost
- left
- budget

## Starter

Ask students:

- What kinds of daily problems can a computer help calculate?

## Concept explanation

Programming becomes useful when we apply calculations to real situations. In this lesson, students use variables and operators together in practical examples.

## Worked example 1

```python
item_price = 250
quantity = 4
total_cost = item_price * quantity

print(f"Total cost: {total_cost}")
```

## Worked example 2

```python
budget = 5000
book_price = 850
books = 2
stationery = 600

spent = (book_price * books) + stationery
left = budget - spent

print(f"Money left: {left}")
```

## Worked example 3

```python
pages_per_day = 20
days = 7
total_pages = pages_per_day * days

print(f"Total pages revised: {total_pages}")
```

## Guided practice

Students write a program for:

- one samosa price
- number of samosas
- total bill

## Independent practice

### Beginner

Find total cost of 5 pencils if one costs 20 rupees.

### Intermediate

A student saves 100 rupees every day for 7 days. Calculate total savings.

### Challenge

A class has 32 students. Each student brings 2 notebooks. How many notebooks total?

## Common mistakes

- not using brackets where needed
- unclear variable names
- printing the wrong variable

## Assessment checkpoint

1. Why do we use variables in calculation programs?
2. What does this line mean: `spent = (book_price * books) + stationery`
3. Write one variable name for a bus fare calculation.

## Exit ticket

Students write one 3-line calculation program.

## Homework

Write a small program about a shopping bill, sports score, or study plan.
