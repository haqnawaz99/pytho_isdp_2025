# Lesson 4: Comparison Operators

## Duration

45 minutes

## Learning objective

Students will be able to compare values using comparison operators and understand the boolean result.

## Key vocabulary

- compare
- true
- false
- boolean

## Starter

Show students:

- `10 > 5`
- `8 == 8`
- `7 < 3`

Ask:

- Which are true?
- Which are false?

## Concept explanation

Comparison operators check a relationship between two values and give a boolean result: `True` or `False`.

- `==` equal to
- `!=` not equal to
- `>` greater than
- `<` less than
- `>=` greater than or equal to
- `<=` less than or equal to

## Worked example 1

```python
print(10 > 5)
print(8 == 8)
```

## Worked example 2

```python
marks = 75
print(marks >= 50)
print(marks < 40)
```

## Worked example 3

```python
age = 13
print(age == 13)
print(age != 15)
```

## Guided practice

Students predict:

```python
print(9 < 4)
print(12 >= 12)
print(5 != 5)
```

## Independent practice

### Beginner

Check whether:

- `15 > 10`
- `20 == 25`

### Intermediate

Use a variable `runs = 54` and compare it with `50`.

### Challenge

Create two variables and print three comparisons.

## Common mistakes

- using `=` instead of `==`
- thinking the result is text instead of boolean
- reading `>=` and `<=` incorrectly

## Assessment checkpoint

1. What does `==` mean?
2. What is the difference between `=` and `==`?
3. What result does a comparison operator give?

## Exit ticket

Students write one comparison that returns `True`.

## Homework

Write five comparison examples and predict whether each is `True` or `False`.
