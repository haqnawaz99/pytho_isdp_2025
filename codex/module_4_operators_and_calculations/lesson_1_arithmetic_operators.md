# Lesson 1: Arithmetic Operators

## Duration

45 minutes

## Learning objective

Students will be able to use `+`, `-`, `*`, and `/` for simple calculations.

## Key vocabulary

- add
- subtract
- multiply
- divide

## Starter

Ask students:

- If one notebook costs 120 rupees and you buy 2, what calculation do you need?
- If you have 500 rupees and spend 150, what calculation do you need?

## Concept explanation

Arithmetic operators help Python do number calculations.

- `+` adds
- `-` subtracts
- `*` multiplies
- `/` divides

## Worked example 1

```python
print(5 + 3)
print(10 - 4)
```

## Worked example 2

```python
print(6 * 4)
print(12 / 3)
```

## Worked example 3

```python
notebook_price = 120
quantity = 3
total_cost = notebook_price * quantity

print(total_cost)
```

## Guided practice

Students predict and then run:

```python
print(20 + 15)
print(50 - 12)
print(7 * 5)
print(18 / 2)
```

## Independent practice

### Beginner

Find:

- `8 + 6`
- `15 - 4`

### Intermediate

Calculate the cost of 4 pens if one pen costs 35 rupees.

### Challenge

Write a small program for:

- budget = 1000
- spent = 275

Print money left.

## Common mistakes

- mixing number values with text by mistake
- expecting division to always return a whole number
- forgetting to store a result in a variable

## Assessment checkpoint

1. What does `*` do?
2. What does `/` do?
3. Write one line that multiplies 25 by 4.

## Exit ticket

Students write one addition and one multiplication example.

## Homework

Write a short program for a simple shopping bill using at least two arithmetic operators.
