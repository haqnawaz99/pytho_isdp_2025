# Module 4 Quiz: Operators and Calculations

## Part A: Multiple Choice

1. Which operator gives the remainder?
   - A. `//`
   - B. `%`
   - C. `**`
   - Answer: B

2. What is the result of `3 * 4`?
   - A. 7
   - B. 12
   - C. 34
   - Answer: B

3. What does `==` do?
   - A. stores a value
   - B. compares two values for equality
   - C. joins strings
   - Answer: B

4. When is `and` true?
   - A. when both sides are true
   - B. when one side is true
   - C. never
   - Answer: A

5. What is the result of `10 + 2 * 3`?
   - A. 36
   - B. 16
   - C. 12
   - Answer: B

## Part B: Short Answer

1. What is the difference between `/` and `//`?
2. What does `%` mean in Python?
3. What is the difference between `=` and `==`?
4. Why do we use brackets in expressions?

## Part C: Coding

1. Write code to calculate the total cost of 4 notebooks at 120 rupees each.
2. Print the result of `17 % 5`.
3. Write one comparison that returns `False`.

## Answer Key

### Part B Suggested Answers

1. `/` gives normal division, while `//` gives whole-number division.
2. It gives the remainder after division.
3. `=` stores a value, while `==` compares values.
4. Brackets make the order of operations clear and can change the result.

### Part C Suggested Solutions

```python
notebook_price = 120
quantity = 4
print(notebook_price * quantity)
```

```python
print(17 % 5)
```

```python
print(6 > 9)
```
