# Python Course for Pakistani School and Madrasa Learners

This folder contains the redesigned beginner Python course built for Pakistani school and Madrasa students.

## Start here

- `COURSE_MAP.md`: full course sequence from Module 1 to Module 9
- `WEEKLY_TEACHING_PLAN.md`: suggested teaching pace across weeks
- `SETUP_GUIDE.md`: practical setup guidance for school labs
- `TEACHER_HANDBOOK.md`: delivery advice, misconceptions, and classroom strategy
- `python_course_redesign_guide.md`: full design rationale and curriculum review

## Module structure

Each module folder contains:

- lesson notes
- practice sheet
- quiz
- runnable code examples

## Modules

1. `module_1_printing_and_strings`
2. `module_2_variables_and_user_input`
3. `module_3_numbers_data_types_and_conversion`
4. `module_4_operators_and_calculations`
5. `module_5_decisions_with_if_statements`
6. `module_6_repetition_with_loops`
7. `module_7_functions`
8. `module_8_lists_and_collections`
9. `module_9_mini_projects`

## Extra support

- `implementation_checklist.md`: rollout checklist
- `advanced_continuation_path.md`: what to teach after the beginner course

## Recommended teaching order

1. Read `SETUP_GUIDE.md`
2. Read `COURSE_MAP.md`
3. Use `WEEKLY_TEACHING_PLAN.md` to schedule lessons
4. Teach modules in order
5. Finish with `module_9_mini_projects`
