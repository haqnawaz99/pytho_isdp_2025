# Module 3 Quiz: Numbers, Data Types, and Type Conversion

## Part A: Multiple Choice

1. Which value is an integer?
   - A. `"15"`
   - B. `15`
   - C. `15.0`
   - Answer: B

2. Which value is a float?
   - A. `12.5`
   - B. `"12.5"`
   - C. `True`
   - Answer: A

3. What does `type()` do?
   - A. joins strings
   - B. checks the data type
   - C. takes user input
   - Answer: B

4. What does `input()` return before conversion?
   - A. `int`
   - B. `float`
   - C. `str`
   - Answer: C

5. Which conversion is correct for `"45.5"`?
   - A. `int("45.5")`
   - B. `float("45.5")`
   - C. `bool("45.5")`
   - Answer: B

## Part B: Short Answer

1. What is the difference between `25` and `"25"`?
2. What is a float?
3. Why do we use `str()`?
4. Why can `int("Ali")` not work?

## Part C: Coding

1. Create a variable called `age` and store an integer.
2. Print the type of `"Karachi"`.
3. Ask the user for a decimal price and store it correctly.

## Answer Key

### Part B Suggested Answers

1. `25` is an integer, while `"25"` is a string.
2. A float is a number with a decimal point.
3. We use `str()` to convert a value into text.
4. Because `Ali` is not a numeric value.

### Part C Suggested Solutions


```python

```
