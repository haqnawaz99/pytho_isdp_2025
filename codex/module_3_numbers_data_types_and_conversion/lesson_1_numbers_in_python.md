# Lesson 1: Numbers in Python

## Duration

40 minutes

## Learning objective

Students will be able to identify whole numbers and decimal numbers in Python.

## Key vocabulary

- number
- integer
- decimal
- float

## Starter

Show students:

- `12`
- `45`
- `3.5`
- `99.99`

Ask:

- Which values are whole numbers?
- Which values have decimals?

## Concept explanation

Python uses different forms for numbers:

- whole numbers are called integers or `int`
- decimal numbers are called floating-point numbers or `float`

Examples from daily life:

- age: `13`
- cricket runs: `78`
- notebook price: `120.5`
- temperature: `36.7`

## Worked example 1

```python
age = 14
print(age)
```

## Worked example 2

```python
temperature = 37.5
print(temperature)
```

## Worked example 3

```python
marks = 88
price = 250.75

print(marks)
print(price)
```

## Guided practice

Ask students to classify:

- `42`
- `42.0`
- `100`
- `12.25`

## Independent practice

### Beginner

Store your age in a variable and print it.

### Intermediate

Store a price in rupees with a decimal and print it.

### Challenge

Store two whole-number values and two decimal values.

## Common mistakes

- putting quotes around number values by mistake
- thinking `42.0` is an integer
- confusing numbers with digits inside strings

## Assessment checkpoint

1. What is an integer?
2. What is a float?
3. Is `17.0` an integer or a float?

## Exit ticket

Students write one integer and one float variable.

## Homework

Write a short program with:

- one age
- one price
- one temperature
