# Student Practice Sheet: Module 3

## Instructions

Read each task carefully. Identify the type first, then write the code.

## Section 1: Integers and Floats

1. Write one integer for age.
2. Write one float for price.
3. Write one integer for cricket runs.
4. Write one float for temperature.

## Section 2: Data Types

Write the type of each:

1. `"Multan"`
2. `25`
3. `7.5`
4. `False`

## Section 3: Using `type()`

Write code that prints the type of:

1. your name
2. `45`
3. `12.75`
4. `True`

## Section 4: Conversion

Write code to:

1. convert `"50"` to integer
2. convert `"8.25"` to float
3. convert `90` to string

## Section 5: Input Conversion

Write code that asks the user for:

1. age as an integer
2. price as a float

## Challenge Section

1. Explain why `int("4.5")` does not work.
2. Fix this code:

```python
marks = int("95.5")
```

3. Ask the user for marks and print the type after conversion.

## Teacher checking guide

Check whether students can:

- tell text and numbers apart
- identify `int`, `float`, `str`, and `bool`
- use `type()`
- convert correctly
- explain simple conversion failures
