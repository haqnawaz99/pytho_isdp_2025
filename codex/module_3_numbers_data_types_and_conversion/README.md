# Module 3: Numbers, Data Types, and Type Conversion

This module introduces number types, basic data types, and safe conversion between text and numbers.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_numbers_in_python.md`: integers and floats
- `lesson_2_basic_data_types.md`: `str`, `int`, `float`, `bool`
- `lesson_3_using_type.md`: checking data type with `type()`
- `lesson_4_type_conversion.md`: `int()`, `float()`, and `str()`
- `lesson_5_input_and_number_conversion.md`: converting input before use
- `lesson_6_conversion_mistakes.md`: common beginner errors
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Teach the six lessons in order.
2. Do not overload students with many operators yet.
3. Emphasize that `input()` returns text unless we convert it.
4. Use real examples from ages, marks, prices, and temperatures.
