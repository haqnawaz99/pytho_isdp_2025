# Lesson 2: Basic Data Types

## Duration

45 minutes

## Learning objective

Students will be able to explain the basic Python data types `str`, `int`, `float`, and `bool`.

## Key vocabulary

- data type
- string
- integer
- float
- boolean

## Starter

Write these on the board:

- `"Lahore"`
- `25`
- `12.5`
- `True`

Ask:

- What kind of value is each one?

## Concept explanation

A data type tells Python what kind of value it is working with.

For beginners, focus on four common types:

- `str` for text
- `int` for whole numbers
- `float` for decimal numbers
- `bool` for `True` or `False`

## Worked example 1

```python
city = "Lahore"
print(city)
```

Teacher note:

- This is text, so it is a string.

## Worked example 2

```python
age = 15
print(age)
```

Teacher note:

- This is a whole number, so it is an integer.

## Worked example 3

```python
price = 85.5
is_present = True

print(price)
print(is_present)
```

Teacher note:

- `price` is a float.
- `is_present` is a boolean.

## Guided practice

Students label each value:

- `"Pakistan"`
- `80`
- `7.25`
- `False`

## Independent practice

### Beginner

Write one example of each type.

### Intermediate

Store city, age, price, and attendance status in four variables.

### Challenge

Write a short comment-free program that shows all four types with meaningful values.

## Common mistakes

- writing `true` instead of `True`
- thinking all numbers are `int`
- confusing the word `False` with a string if quotes are used

## Assessment checkpoint

1. What is a data type?
2. What are the two possible boolean values?
3. Is `"False"` a boolean or a string?

## Exit ticket

Students identify the type of three teacher-given values.

## Homework

Make a 4-line program using `str`, `int`, `float`, and `bool`.
