# Module 8: Lists and Collections

This module introduces lists first and then gives a careful beginner-level introduction to tuples, sets, and dictionaries.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_introduction_to_lists.md`: what a list is
- `lesson_2_accessing_list_items.md`: indexes and reading list items
- `lesson_3_modifying_and_updating_lists.md`: append, change, and remove
- `lesson_4_looping_through_lists.md`: using loops with lists
- `lesson_5_tuples_and_sets_intro.md`: simple contrast with lists
- `lesson_6_dictionaries_intro.md`: key-value storage
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Spend most of the time on lists.
2. Use tuples, sets, and dictionaries as careful introductions, not advanced deep dives.
3. Keep examples small and concrete.
4. Use many real-life item collections from Pakistani student life.
