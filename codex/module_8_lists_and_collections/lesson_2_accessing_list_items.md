# Lesson 2: Accessing List Items

## Duration

45 minutes

## Learning objective

Students will be able to access list items using indexes.

## Key vocabulary

- index
- first item
- position

## Starter

Show:

```python
cities = ["Lahore", "Karachi", "Multan"]
```

Ask:

- How can we get only `Karachi`?

## Concept explanation

Each item in a list has an index. Indexing starts at 0.

For this list:

- `"Lahore"` is index 0
- `"Karachi"` is index 1
- `"Multan"` is index 2

## Worked example 1

```python
cities = ["Lahore", "Karachi", "Multan"]
print(cities[0])
print(cities[1])
```

## Worked example 2

```python
foods = ["Biryani", "Samosa", "Mango"]
print(foods[2])
```

## Worked example 3

```python
subjects = ["Math", "Urdu", "Science"]
print(subjects[0])
print(subjects[2])
```

## Guided practice

Students label indexes for:

```python
books = ["Quran", "Math", "English"]
```

## Independent practice

### Beginner

Print the first item from a list of fruits.

### Intermediate

Print the second city from a city list.

### Challenge

Create a list of four items and print the first and last visible item by index.

## Common mistakes

- starting indexes at 1
- using round brackets instead of square brackets
- choosing an index that does not exist

## Assessment checkpoint

1. What is the first index in a list?
2. What does `cities[1]` mean?
3. Why does indexing matter?

## Exit ticket

Students write one list and access one item from it.

## Homework

Create two lists and print selected items from each.
