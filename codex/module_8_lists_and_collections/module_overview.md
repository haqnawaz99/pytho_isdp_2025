# Module 8 Overview: Lists and Collections

## Module title

Lists and Collections

## Recommended level

Beginner students who have completed Module 7

## Duration

6 lessons, 40 to 45 minutes each

## Module purpose

This module teaches students how to store multiple values together in Python. Students first learn lists well, then get a simple introduction to tuples, sets, and dictionaries so they can recognize different ways of organizing data.

## Learning objectives

By the end of this module, students will be able to:

- create a list
- access list items using indexes
- add, change, and remove list items
- use loops with lists
- explain the basic idea of tuples and sets
- create and use a simple dictionary

## Prior knowledge

Students should be able to:

- use variables
- write loops
- write functions
- print values clearly

## Module progression

### Lesson 1

- Focus: what a list is
- Outcome: students can store multiple values in one list

### Lesson 2

- Focus: accessing list items
- Outcome: students can read values by index

### Lesson 3

- Focus: modifying lists
- Outcome: students can add, change, and remove items

### Lesson 4

- Focus: looping through lists
- Outcome: students can use loops to process all items

### Lesson 5

- Focus: tuples and sets
- Outcome: students understand the simple difference between these collection types

### Lesson 6

- Focus: dictionaries
- Outcome: students can use key-value pairs

## Key vocabulary

- list
- index
- item
- append
- tuple
- set
- dictionary

## Pakistan-relevant teaching contexts

- city names
- subject lists
- shopping items
- cricket teams or scores
- school timetables
- prayer names as labels
- books and notebooks
- student mark records

## Teacher reminders

- Teach lists thoroughly before moving to other collection types.
- Trace indexes carefully on the board.
- Emphasize that indexing starts at 0.
- Keep dictionary examples simple and shallow.
- Do not introduce list comprehensions here.

## Common misconceptions

- Students may think the first index is 1.
- Students may confuse an item with its index.
- Students may think sets keep duplicates and order like lists.
- Students may think dictionary keys are positions like indexes.

## End-of-module evidence

A student can complete this module successfully if they can:

- create and print a list
- access items with indexes
- update a list with `append()` or assignment
- use a `for` loop with list items
- explain a simple tuple or set example
- create a small dictionary and read one value from it
