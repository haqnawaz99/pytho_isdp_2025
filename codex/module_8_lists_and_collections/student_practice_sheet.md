# Student Practice Sheet: Module 8

## Instructions

Read each task carefully. Create the collection, then print the result.

## Section 1: Lists

1. Create a list of three cities.
2. Create a list of three fruits.
3. Create a list of three school subjects.

## Section 2: Accessing Items

1. Print the first city in your city list.
2. Print the second fruit in your fruit list.
3. Print the third subject in your subject list.

## Section 3: Updating Lists

1. Add one city to a city list.
2. Change one fruit in a fruit list.
3. Remove one subject from a subject list.

## Section 4: Looping Through Lists

1. Print each city in a city list.
2. Print each fruit with the message `I like ___`.
3. Print each subject with the message `Study ___`.

## Section 5: Tuples and Sets

1. Create a tuple of three weekdays.
2. Create a set of fruits with one repeated fruit.
3. Explain what happens to the repeated fruit.

## Section 6: Dictionaries

1. Create a dictionary with two subjects and marks.
2. Create a dictionary with name and city.
3. Print one value from each dictionary.

## Challenge Section

1. Create a shopping list and loop through it.
2. Create a student dictionary and print all stored values one by one.
3. Explain the difference between list, tuple, set, and dictionary.

## Teacher checking guide

Check whether students can:

- create lists correctly
- access items with indexes
- update a list safely
- use loops with lists
- describe tuples and sets simply
- use dictionary keys correctly
