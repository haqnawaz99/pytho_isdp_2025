# Lesson 1: Introduction to Lists

## Duration

45 minutes

## Learning objective

Students will be able to explain what a list is and create a simple list in Python.

## Key vocabulary

- list
- item
- collection

## Starter

Ask students:

- If you want to store three city names in one place, what could Python use?

## Concept explanation

A list stores multiple values in one variable. Items are written inside square brackets and separated by commas.

Examples:

- cities
- foods
- subjects

## Worked example 1

```python
cities = ["Lahore", "Karachi", "Multan"]
print(cities)
```

## Worked example 2

```python
foods = ["Biryani", "Samosa", "Mango"]
print(foods)
```

## Worked example 3

```python
subjects = ["Math", "Urdu", "Science"]
print(subjects)
```

## Guided practice

Students create a list of:

- three favorite fruits

## Independent practice

### Beginner

Create a list of three cities.

### Intermediate

Create a list of three school subjects.

### Challenge

Create a list of five shopping items.

## Common mistakes

- forgetting square brackets
- forgetting commas between items
- thinking each item needs a separate variable

## Assessment checkpoint

1. What is a list?
2. What symbols are used to make a list?
3. Why is a list useful?

## Exit ticket

Students write one list from memory.

## Homework

Create three different lists from daily life.
