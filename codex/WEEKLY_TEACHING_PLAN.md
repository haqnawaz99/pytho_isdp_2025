# Weekly Teaching Plan

## Recommended course length

19 to 23 teaching weeks

## Standard pace

- 2 lessons per week in schools with steady access
- 1 lesson per week in slower or mixed-resource environments

## Suggested schedule

### Weeks 1-2

- Module 1: Printing and Strings

### Weeks 3-4

- Module 2: Variables and User Input

### Weeks 5-6

- Module 3: Numbers, Data Types, and Type Conversion

### Weeks 7-8

- Module 4: Operators and Calculations

### Weeks 9-11

- Module 5: Decisions with If Statements

### Weeks 12-14

- Module 6: Repetition with Loops

### Weeks 15-16

- Module 7: Functions

### Weeks 17-19

- Module 8: Lists and Collections

### Weeks 20-23

- Module 9: Mini Projects

## Flexible pacing advice

- If typing speed is weak, add extra revision time after Modules 2, 5, and 6.
- If students are struggling with conditions, delay multi-branch projects.
- If computer access is limited, use pair programming and projector demos.

## Assessment rhythm

- end of each module: quiz + short coding checkpoint
- end of Module 5: mid-course review
- end of Module 9: project showcase
