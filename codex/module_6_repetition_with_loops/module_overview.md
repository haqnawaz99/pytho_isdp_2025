# Module 6 Overview: Repetition with Loops

## Module title

Repetition with Loops

## Recommended level

Beginner students who have completed Module 5

## Duration

6 lessons, 40 to 45 minutes each

## Module purpose

This module teaches students how to repeat tasks in Python without writing the same line again and again. Students learn to count with `for` loops, repeat while a condition is true with `while` loops, and use loops in short real-life programs.

## Learning objectives

By the end of this module, students will be able to:

- explain why loops are useful
- use `for` loops with `range()`
- loop through simple items such as names or cities
- use `while` loops with a stopping condition
- update a counter inside a loop
- write simple loop-based programs

## Prior knowledge

Students should be able to:

- use `if` statements
- compare numbers
- use variables and input
- understand indentation

## Module progression

### Lesson 1

- Focus: the idea of repetition
- Outcome: students understand why loops are needed

### Lesson 2

- Focus: `for` loops with `range()`
- Outcome: students can repeat code a known number of times

### Lesson 3

- Focus: `for` loops with items
- Outcome: students can loop through simple collections

### Lesson 4

- Focus: `while` loops
- Outcome: students can repeat while a condition remains true

### Lesson 5

- Focus: counters and totals
- Outcome: students can update values inside a loop

### Lesson 6

- Focus: real-life loop programs
- Outcome: students can apply loops in small practical tasks

- daily expenses

- weather observations

## Teacher reminders

- Teach loops as a way to save effort.
- Make students trace each loop line by line.
- Use small numbers first.
- Do not rush from `for` to `while`.
- Repeat that indentation is still essential.

## Common misconceptions

- Students may think the loop runs only once.
- Students may forget to update a counter in a `while` loop.
- Students may confuse `range(5)` with printing the number 5 only.
- Students may not understand when the loop stops.

## End-of-module evidence

A student is ready for Module 7 if they can:

- write a `for` loop with `range()`
- write a `while` loop with a stopping condition
- print repeated messages or values in order
- update a counter correctly
- explain how and when a loop stops
