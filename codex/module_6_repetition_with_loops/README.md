# Module 6: Repetition with Loops

This module teaches students how to repeat actions using `for` and `while` loops.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_why_loops.md`: understanding repetition
- `lesson_2_for_loops_with_range.md`: counting with `for` and `range()`
- `lesson_3_for_loops_with_items.md`: looping through simple collections
- `lesson_4_while_loops.md`: repetition with a condition
- `lesson_5_counters_and_totals.md`: simple accumulation ideas
- `lesson_6_real_life_loop_programs.md`: practical loop-based programs
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Start with the idea of repetition before syntax.
2. Teach `for` loops before `while` loops.
3. Use tracing and dry runs often.
4. Warn about infinite loops in a simple, non-frightening way.
