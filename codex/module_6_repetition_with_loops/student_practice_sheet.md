# Student Practice Sheet: Module 6

## Instructions

Predict the output first. Then write and run the code.

## Section 1: Repetition

1. Print `Practice Python` 4 times.
2. Print `Study well` 3 times.

## Section 2: `for` with `range()`

1. Print numbers from 1 to 5.
2. Print numbers from 2 to 6.
3. Print a message 5 times.

## Section 3: `for` with Items

Use a loop to print each item in:

1. `["Lahore", "Karachi", "Quetta"]`
2. `["Mango", "Banana", "Apple"]`
3. `["Math", "Urdu", "Science"]`

## Section 4: `while` Loops

1. Print numbers from 1 to 4 using `while`.
2. Print `Read more` 5 times using `while`.
3. Count from 2 to 8 by 2.

## Section 5: Counters and Totals

1. Add numbers from 1 to 5.
2. Add 10 rupees for 6 days.
3. Use a `while` loop to add 3 four times.

## Section 6: Real-Life Programs

1. Print a study reminder for 7 days.
2. Track total pages read if a student reads 2 pages daily for 5 days.
3. Print cricket over numbers from 1 to 6.

## Challenge Section

1. Write one `for` loop and one `while` loop for the same task.
2. Create a weekly expense tracker using a loop.
3. Explain how a loop stops.

## Teacher checking guide

Check whether students can:

- use loops to reduce repeated code
- understand `range()`
- control a `while` loop safely
- update counters and totals correctly
- apply loops in real contexts
