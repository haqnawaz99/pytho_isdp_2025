# Lesson 2: Strings and Quotes

## Duration

40 to 45 minutes

## Learning objective

Students will be able to identify a string and use single or double quotes correctly.

## Key vocabulary

- string
- quotes
- character

## Starter

Show students:

```python
print("Lahore")
print(25)
```

Ask:

- Which one is text?
- Which one is a number?

## Concept explanation

A string is text in Python. Text must be written inside quotes. A string can be one letter, one word, or a full sentence.

Examples of strings:

- `"Ali"`
- `"Karachi"`
- `"I am learning Python"`

Examples that are not strings:

- `45`
- `12.5`
- `True`

## Worked example 1

```python
print("Lahore")
print("Multan")
```

## Worked example 2

```python
print('Biryani')
print("Samosa")
```

Teacher note:

- Explain that both single and double quotes work.

## Worked example 3

```python
print("I'm ready to learn")
print('He said "Welcome"')
```

Teacher note:

- Show why quote choice matters.

## Guided practice

Ask students to classify each as string or not string:

- `"Quran"`
- `100`
- `"Class 7"`
- `False`

## Independent practice

### Beginner

Write two strings about your favorite food and city.

### Intermediate

Print a sentence with an apostrophe, such as:

- `I'm in Class 8`

### Challenge

Write one line using single quotes and one line using double quotes.

## Common mistakes

- mixing opening and closing quotes incorrectly
- forgetting that numbers without quotes are not strings
- confusing words in output with variable names, even though variables are not introduced yet

## Assessment checkpoint

1. What is a string?
2. Why do we use quotes around text?
3. Is `"25"` a string or a number?

## Exit ticket

Students write one correct string containing their school name.

## Homework

Make a list of five strings from daily life in Pakistan.
