# Lesson 3: Joining Strings

## Duration

45 minutes

## Learning objective

Students will be able to join strings using `+` and include spaces correctly.

## Key vocabulary

- concatenate
- join
- space

## Starter

Write this on the board:

```python
print("Eid" + "Mubarak")
```

Ask students:

- What output do you expect?
- What problem do you notice?

## Concept explanation

When we join strings with `+`, Python sticks them together. This is called concatenation. If we want a space between words, we must add a string that contains a space.

Simple Urdu support:

- concatenate = do text hissay jorna

## Worked example 1

```python
print("Eid" + " " + "Mubarak")
```

## Worked example 2

```python
print("Class" + " " + "7")
print("Cricket" + " " + "Match" + " " + "Today")
```

## Worked example 3

```python
print("My city is " + "Peshawar")
print("My favorite food is " + "biryani")
```

## Guided practice

Students fix the following:

```python
print("Assalamu" + "Alaikum")
print("Lahore" + "Pakistan")
```

## Independent practice

### Beginner

Join:

- `My name is`
- your name

### Intermediate

Join:

- `I study in`
- your class

### Challenge

Create a line that prints:

- `Today we will learn Python`

using at least three joined strings.

## Common mistakes

- forgetting the space string `" "`
- using commas instead of `+` before understanding the difference
- missing one quote or plus sign

## Assessment checkpoint

1. What does `+` do between two strings?
2. Why do we use `" "` in concatenation?
3. Write one line to print `Good Morning Pakistan`.

## Exit ticket

Students create one joined message about their city or school.

## Homework

Write five joined string messages about family, school, or food.
