# Lesson 1: Printing Messages

## Duration

40 minutes

## Learning objective

Students will be able to use `print()` to display text on the screen.

## Key vocabulary

- `print()`
- output
- text

## Starter

Ask students:

- What is a computer program?
- What do you think happens when we tell a computer to show a message?

## Concept explanation

Python uses `print()` to show output on the screen. The word inside the brackets tells Python what message to display.

Simple Urdu support:

- output = nateeja jo screen par nazar aata hai

## Worked example 1

```python
print("Hello")
```

Teacher note:

- Explain that the computer shows the word `Hello` on the screen.

## Worked example 2

```python
print("Assalamu Alaikum")
print("Welcome to Python class")
```

Teacher note:

- Show that each `print()` line creates a new output line.

## Worked example 3

```python
print("My name is Ahmed")
print("I live in Lahore")
print("I like mango")
```

## Guided practice

Students type and run:

```python
print("Pakistan")
print("Python is fun")
print("Today is a good day to learn")
```

## Independent practice

### Beginner

Print:

- your name
- your city

### Intermediate

Print:

- your school or madrasa name
- your favorite subject

### Challenge

Print three sentences about yourself.

## Common mistakes

- forgetting brackets in `print()`
- forgetting quotes around text
- writing plain text without a command

## Assessment checkpoint

1. What does `print()` do?
2. How many lines will this code print?

```python
print("A")
print("B")
```

3. Write one line of code to print `I love learning`.

## Exit ticket

Students write one working `print()` line without copying from the board.

## Homework

Write a short Python program with four `print()` lines about your daily routine.
