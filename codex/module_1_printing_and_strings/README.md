# Module 1: Printing and Strings

This module is the first teaching unit in the redesigned Python course.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_printing_messages.md`: first lesson on `print()`
- `lesson_2_strings_and_quotes.md`: strings, quotes, and spaces
- `lesson_3_joining_strings.md`: concatenation and simple message building
- `lesson_4_length_and_string_methods.md`: `len()` and beginner string methods
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Teach the four lessons in order.
2. Use the code examples during live demonstration.
3. Give the practice sheet after Lesson 3 or Lesson 4.
4. Use the quiz as the end-of-module checkpoint.
