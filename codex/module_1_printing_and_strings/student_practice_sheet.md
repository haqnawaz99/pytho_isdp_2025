# Student Practice Sheet: Module 1

## Instructions

Write your answers in Python. First predict the output, then run the code.

## Section 1: Printing

1. Print your name.
2. Print your school or madrasa name.
3. Print your favorite fruit.

## Section 2: Strings

Which of the following are strings?

1. `"Lahore"`
2. `56`
3. `"Class 6"`
4. `True`
5. `"Biryani"`

## Section 3: Joining Strings

Write code to print:

1. `My city is Karachi`
2. `Welcome Class 7`
3. `Pakistan Cricket Team`

## Section 4: Length

Use `len()` to find the length of:

1. `"Ali"`
2. `"Islamabad"`
3. `"my school"`

## Section 5: String Methods

Write code for each:

1. Change `"pakistan"` to uppercase.
2. Change `"eid mubarak"` to title case.
3. Remove extra spaces from `"  mango  "`.

## Challenge Section

1. Print your full name in uppercase.
2. Print the length of your favorite subject.
3. Join three strings to make a sentence about your city.

## Teacher checking guide

Check whether students can:

- use quotes correctly
- use `print()` correctly
- join strings with spaces
- use `len()` correctly
- call simple string methods with brackets
