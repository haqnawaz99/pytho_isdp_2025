# Module 1 Quiz: Printing and Strings

## Part A: Multiple Choice

1. Which command shows output on the screen?
   - A. `input()`
   - B. `print()`
   - C. `len()`
   - Answer: B

2. Which of the following is a string?
   - A. `45`
   - B. `False`
   - C. `"45"`
   - Answer: C

3. What does `+` do between strings?
   - A. subtracts
   - B. joins
   - C. compares
   - Answer: B

4. What is the output of `len("Ali")`?
   - A. 2
   - B. 3
   - C. 4
   - Answer: B

5. What does `.upper()` do?
   - A. removes spaces
   - B. changes text to capital letters
   - C. counts letters
   - Answer: B

## Part B: Short Answer

1. What is a string?
2. Why do we use quotes around text?
3. What is the purpose of `" "` in string joining?
4. What does `.strip()` do?

## Part C: Coding

1. Write code to print your name and city on two separate lines.
2. Join the words `Good`, `Morning`, and `Pakistan` into one line.
3. Print the length of the string `"Islamabad"`.

## Answer Key

### Part B Suggested Answers

1. A string is text written inside quotes.
2. Quotes tell Python that the value is text.
3. It adds a space between joined words.
4. It removes spaces from the beginning and end of a string.

### Part C Suggested Solutions

```python
print("Quetta")
```

```python

```
