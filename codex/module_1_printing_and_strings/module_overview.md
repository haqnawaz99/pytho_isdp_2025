# Module 1 Overview: Printing and Strings

## Module title

Printing and Strings

## Recommended level

Beginner students, roughly ages 11 to 16

## Duration

4 lessons, 40 to 45 minutes each

## Module purpose

This module gives students their first successful experience with Python. They learn how to print messages, understand what a string is, join pieces of text, and make small changes to text using simple string tools.

## Learning objectives

By the end of this module, students will be able to:

- run a simple Python program
- use `print()` to display output
- identify text as a string
- use quotes correctly around text
- join strings using `+`
- count characters using `len()`
- apply simple string methods such as `.upper()`, `.lower()`, `.title()`, and `.strip()`

## Prior knowledge

Students do not need prior coding experience.

## Module progression

### Lesson 1

- Focus: `print()`
- Outcome: students can display messages on the screen

### Lesson 2

- Focus: strings and quotes
- Outcome: students can recognize strings and write them correctly

### Lesson 3

- Focus: concatenation
- Outcome: students can combine text to make short messages

### Lesson 4

- Focus: `len()` and string methods
- Outcome: students can inspect and improve text output

## Teacher reminders

- Keep the pace slow in this first module.
- Let students predict the output before running each example.
- Avoid long theory explanations.
- If students are weak in English, explain the idea in simple English and support with short Urdu clarifications.
- Focus on correctness of syntax more than speed.

## Common misconceptions

- Students may think quotes are optional around text.
- Students may confuse the output on screen with the code they typed.
- Students often forget spaces when joining strings.
- Students may think `len()` counts words instead of characters.

## End-of-module evidence

A student is ready for Module 2 if they can:

- print three correct lines of text
- join two or three strings into one message
- explain what a string is
- use `len()` on a simple string
- fix a missing-quote or missing-space mistake
