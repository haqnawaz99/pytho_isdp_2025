# Course Map

## Course title

Beginner Python for Pakistani School and Madrasa Learners

## Course flow

### Module 1: Printing and Strings

- Lessons: 4
- Focus: output, strings, concatenation, `len()`, simple string methods

### Module 2: Variables and User Input

- Lessons: 5
- Focus: variables, assignment, naming, `input()`, `f-strings`

### Module 3: Numbers, Data Types, and Type Conversion

- Lessons: 6
- Focus: `int`, `float`, `str`, `bool`, `type()`, conversion

### Module 4: Operators and Calculations

- Lessons: 6
- Focus: arithmetic, comparison, logical operators, precedence

### Module 5: Decisions with If Statements

- Lessons: 6
- Focus: `if`, `if-else`, `if-elif-else`, input checks, validation

### Module 6: Repetition with Loops

- Lessons: 6
- Focus: `for`, `range()`, `while`, counters, totals

### Module 7: Functions

- Lessons: 6
- Focus: define, call, parameters, return values, defaults

### Module 8: Lists and Collections

- Lessons: 6
- Focus: lists, indexes, updates, loops with lists, intro to tuples, sets, dictionaries

### Module 9: Mini Projects

- Lessons: 6
- Focus: integration of course concepts through small complete programs

## Core teaching sequence

`print()` -> strings -> variables -> input -> numbers -> conversion -> operators -> conditions -> loops -> functions -> lists -> projects

## Recommended delivery principle

- teach one main new concept per lesson
- revise previous concepts briefly every lesson
- keep code examples local and relatable
- prefer short working programs over long abstract explanations
