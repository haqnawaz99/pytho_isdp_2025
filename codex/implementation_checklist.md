# Implementation Checklist

## Immediate Fixes

- Remove duplicated topic coverage across `Variables`, `Data Types`, and `Operators`
- Standardize file encoding to UTF-8 to remove broken characters
- Replace emoji-heavy classroom scripts with cleaner lesson notes and short runnable examples
- Rewrite lessons that combine too many new ideas at once

## Course Structure Tasks

- Adopt the redesigned module sequence from `python_course_redesign_guide.md`
- Set a standard lesson format for every unit
- Separate core lessons, practice tasks, quizzes, and solutions
- Add learning objectives and checkpoints to every lesson

## Pakistan-Relevant Improvements

- Balance Islamic examples with school, family, sports, shopping, weather, and agriculture examples
- Add Urdu support words only where they improve clarity
- Use respectful, inclusive language for both school and Madrasa settings

## Technical Standards

- Target Python 3.11+ in course notes
- Prefer simple, readable code over clever shortcuts
- Delay advanced topics like comprehensions until students are ready
- Keep one concept focus per lesson

## Teacher Rollout

- Pilot the first three modules with one student group
- Track where learners struggle with input, variables, and `if` statements
- Adjust lesson timing after the pilot
- Convert the remaining old material module by module, not all at once
