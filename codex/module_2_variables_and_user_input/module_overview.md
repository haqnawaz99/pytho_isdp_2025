# Module 2 Overview: Variables and User Input

## Module title

Variables and User Input

## Recommended level

Beginner students who have completed Module 1

## Duration

5 lessons, 40 to 45 minutes each

## Module purpose

This module teaches students how to store information in variables and use that information later in a program. It also introduces user input so students can make small interactive programs.

## Learning objectives

By the end of this module, students will be able to:

- explain what a variable is in simple words
- assign text values to variables using `=`
- choose simple and meaningful variable names
- use variables inside `print()` statements
- take input from the user using `input()`
- build short personalized messages
- use a basic `f-string` to display variable values

## Prior knowledge

Students should be able to:

- use `print()`
- identify a string
- write text inside quotes

## Module progression

### Lesson 1

- Focus: what a variable is
- Outcome: students understand that a variable stores a value

### Lesson 2

- Focus: variable naming and assignment
- Outcome: students can create correctly named variables

### Lesson 3

- Focus: user input
- Outcome: students can collect text from the user

### Lesson 4

- Focus: using variables in output
- Outcome: students can print messages using stored values

### Lesson 5

- Focus: first use of `f-strings`
- Outcome: students can format simple messages more clearly

## Teacher reminders

- Explain a variable as a named box or label for stored information.
- Keep examples text-based first. Do not bring in number conversion yet.
- Remind students that `input()` gives text.
- Delay more advanced formatting until Lesson 5.
- Spend time showing the difference between variable names and string values.

## Common misconceptions

- Students may think `=` means "is equal to" in the math sense instead of "store this value".
- Students may put quotes around variable names when they want the value.
- Students may forget to save `input()` in a variable.
- Students may think the prompt text is the stored value.

## End-of-module evidence

A student is ready for Module 3 if they can:

- create and use 2 to 3 variables correctly
- ask the user for text input
- print a personalized message using stored input
- explain the difference between `name` and `"name"`
- write one simple `f-string`
