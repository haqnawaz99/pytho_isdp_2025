# Module 2: Variables and User Input

This module builds on Module 1 and introduces students to storing information and taking simple input from the user.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_what_is_a_variable.md`: introducing variables
- `lesson_2_variable_names_and_assignment.md`: naming rules and storing values
- `lesson_3_user_input.md`: using `input()`
- `lesson_4_using_variables_in_output.md`: combining variables in messages
- `lesson_5_f_strings_intro.md`: first introduction to `f-strings`
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Teach the five lessons in order.
2. Keep typing tasks short in the early lessons.
3. Use `f-strings` only after students are comfortable with basic variables and input.
4. End with the practice sheet and quiz.
