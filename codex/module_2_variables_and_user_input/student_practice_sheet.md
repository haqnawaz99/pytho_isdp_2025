# Student Practice Sheet: Module 2

## Instructions

Write your answers in Python. Read each task carefully and use meaningful variable names.

## Section 1: Simple Variables

1. Store your name in a variable and print it.
2. Store your city in a variable and print it.
3. Store your favorite food in a variable and print it.

## Section 2: Good Variable Names

Write good variable names for:

1. school name
2. favorite sport
3. student class
4. favorite subject

## Section 3: User Input

Write code that asks the user for:

1. their name
2. their city
3. their favorite subject

## Section 4: Output Messages

Write code that prints:

1. `Welcome ___`
2. `My city is ___`
3. `I like ___`

Use variables or user input.

## Section 5: f-Strings

Write an `f-string` for each:

1. `Ali lives in Karachi.`
2. `Ayesha likes mango.`
3. `Hamza studies in Class 7.`

## Challenge Section

Write a short Python program that asks for:

- name
- school or madrasa
- city

Then prints a 2-line introduction.

## Teacher checking guide

Check whether students can:

- assign values correctly with `=`
- avoid spaces in variable names
- store input in variables
- use variables in output
- write a simple `f-string`
