# Lesson 1: What Is a Variable?

## Duration

40 minutes

## Learning objective

Students will be able to explain that a variable stores a value and use one simple text variable in Python.

## Key vocabulary

- variable
- value
- store

## Starter

Ask students:

- If you need to remember a student's name in a program, where can you keep it?
- Why should a program store information instead of writing everything again and again?

## Concept explanation

A variable is a named place that stores a value. A helpful simple explanation is:

- variable = aik naam wala box

The name is written on the left side, and the value is written on the right side.

## Worked example 1

```python
name = "Ali"
print(name)
```

Teacher note:

- `name` is the variable name.
- `"Ali"` is the value stored in it.

## Worked example 2

```python
city = "Lahore"
print(city)
```

## Worked example 3

```python
greeting = "Assalamu Alaikum"
print(greeting)
```

## Guided practice

Students type and run:

```python
food = "Biryani"
print(food)
```

## Independent practice

### Beginner

Store your name in a variable and print it.

### Intermediate

Store your city in a variable and print it.

### Challenge

Store your school or madrasa name in a variable and print it.

## Common mistakes

- forgetting quotes around text values
- writing `print("name")` when the goal is to print the variable value
- using spaces in the variable name

## Assessment checkpoint

1. What is a variable?
2. In `name = "Ali"`, what is the variable name?
3. In `name = "Ali"`, what is the value?

## Exit ticket

Students write one working variable example from memory.

## Homework

Write a 3-line program that stores and prints:

- your name
- your city
- your favorite food
