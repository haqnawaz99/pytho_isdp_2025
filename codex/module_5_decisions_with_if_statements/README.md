# Module 5: Decisions with If Statements

This module introduces conditional logic so students can make programs choose between different actions.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_basic_if_statements.md`: simple `if`
- `lesson_2_if_else_statements.md`: two-way decisions
- `lesson_3_if_elif_else.md`: multi-way decisions
- `lesson_4_comparing_user_input.md`: input with conditions
- `lesson_5_simple_validation.md`: basic validation and checking
- `lesson_6_real_life_decision_programs.md`: practical conditional programs
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Teach one new branching pattern at a time.
2. Trace each condition aloud before running the program.
3. Emphasize indentation and the colon `:`.
4. Keep conditions simple before combining them in later modules.
