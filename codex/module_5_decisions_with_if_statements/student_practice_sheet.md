# Student Practice Sheet: Module 5

## Instructions

Read each task carefully. First predict what the program should do, then write the code.

## Section 1: Basic `if`

1. Print `Good score` if marks are 80 or more.
2. Print `Hot day` if temperature is above 35.
3. Print `Big score` if cricket runs are above 60.

## Section 2: `if-else`

1. If age is 18 or more, print `Adult`, otherwise print `Minor`.
2. If rainfall is above 0, print `Rainy`, otherwise print `Dry`.
3. If budget is 500 or more, print `Can buy`, otherwise print `Cannot buy`.

## Section 3: `if-elif-else`

1. Classify marks as:
   - `Excellent` for 80 or more
   - `Pass` for 50 to 79
   - `Fail` otherwise

2. Classify temperature as:
   - `Very hot` for 40 or more
   - `Hot` for 30 to 39
   - `Pleasant` otherwise

## Section 4: Input and Conditions

1. Ask the user for marks and print pass or fail.
2. Ask the user for cricket runs and print half century or below fifty.
3. Ask the user for city and print one message for `Lahore` and another for other cities.

## Section 5: Validation

1. Check whether marks are between 0 and 100.
2. Check whether age is zero or more.
3. Check whether rainfall is zero or more.

## Challenge Section

1. Write a complete result-checking program with input.
2. Write a weather advice program with 3 outcomes.
3. Write a small shopping budget decision program.

## Teacher checking guide

Check whether students can:

- write conditions correctly
- use `==` instead of `=`
- indent code properly
- choose correct branch order
- validate input sensibly
