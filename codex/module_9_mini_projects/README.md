# Module 9: Mini Projects

This module helps students combine earlier Python concepts into complete beginner-friendly projects.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_project_skills_review.md`: reviewing the building blocks needed for projects
- `lesson_2_project_school_canteen_bill.md`: billing and totals project
- `lesson_3_project_student_report_card.md`: input, conditions, and functions project
- `lesson_4_project_cricket_score_helper.md`: sports-based logic project
- `lesson_5_project_weather_and_rainfall_helper.md`: local-context decision project
- `lesson_6_project_showcase_and_reflection.md`: presentation and reflection
- `project_rubric.md`: student project rubric
- `teacher_project_guidance.md`: delivery guidance and differentiation notes
- `student_practice_sheet.md`: project planning worksheet
- `code_examples/`: starter and sample project files

Recommended use:

1. Start with a review of earlier skills before full project building.
2. Use one shared class project first, then move to individual or pair work.
3. Focus on readable code, not feature overload.
4. Use the rubric to assess understanding, not only whether the code runs.
5. Use the `project_*_starter.py` files first, then compare with `project_*_solution.py`.
