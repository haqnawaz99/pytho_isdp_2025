# Lesson 2: Defining and Calling Simple Functions

## Duration

45 minutes

## Learning objective

Students will be able to define a simple function and call it.

## Key vocabulary

- `def`
- call
- function name

## Starter

Show:

```python
def say_hello():
    print("Hello")
```

Ask:

- What is still missing if we want the message to appear?

## Concept explanation

To use a function, we must:

1. define it
2. call it

Calling a function means writing its name with brackets.

## Worked example 1

```python
def say_hello():
    print("Hello")

say_hello()
```

## Worked example 2

```python
def school_message():
    print("Study hard every day")

school_message()
school_message()
```

## Worked example 3

```python
def show_food():
    print("Biryani")

show_food()
```

## Guided practice

Students write a function called `say_welcome()` and call it twice.

## Independent practice

### Beginner

Define a function that prints your city.

### Intermediate

Define a function that prints a school reminder and call it three times.

### Challenge

Define two different functions and call both.

## Common mistakes

- forgetting the brackets when calling
- placing the call inside the function body by accident
- incorrect indentation

## Assessment checkpoint

1. What is the difference between defining and calling?
2. How do we call a function?
3. How many times will a function run if called twice?

## Exit ticket

Students write one function and one function call.

## Homework

Write two simple functions and call each one at least once.
