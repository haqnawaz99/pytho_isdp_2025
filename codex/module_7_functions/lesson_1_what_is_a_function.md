# Lesson 1: What Is a Function?

## Duration

40 minutes

## Learning objective

Students will be able to explain what a function is and why functions are useful.

## Key vocabulary

- function
- reuse
- task

## Starter

Show repeated code:

```python
print("Welcome to class")
print("Welcome to class")
print("Welcome to class")
```

Ask:

- Can we give this repeated task a name?

## Concept explanation

A function is a named block of code that does a specific job. We can write it once and use it many times.

Simple comparison:

- a function is like a reusable instruction set
- it helps keep code organized

## Worked example 1

```python
def say_welcome():
    print("Welcome to class")
```

Teacher note:

- Explain that this defines the function, but does not run it yet.

## Worked example 2

```python
def say_salam():
    print("Assalamu Alaikum")
```

## Worked example 3

```python
def show_city():
    print("Lahore")
```

## Guided practice

Ask students:

- What task does each function name describe?

## Independent practice

### Beginner

Write a function name for printing your school name.

### Intermediate

Write a function name for showing a study reminder.

### Challenge

Suggest three useful function names for a student program.

## Common mistakes

- thinking the function runs during definition
- using unclear names
- forgetting the colon after `def`

## Assessment checkpoint

1. What is a function?
2. Why do we use functions?
3. Does writing `def` run the function immediately?

## Exit ticket

Students explain one reason functions are useful.

## Homework

Write three examples of tasks that could become functions in a Python program.
