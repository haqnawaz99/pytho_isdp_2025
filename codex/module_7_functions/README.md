# Module 7: Functions

This module introduces functions as reusable blocks of code.

Contents:

- `module_overview.md`: module goals, progression, timing, and teacher notes
- `lesson_1_what_is_a_function.md`: understanding functions and calling them
- `lesson_2_defining_simple_functions.md`: creating simple functions
- `lesson_3_functions_with_parameters.md`: passing information into functions
- `lesson_4_functions_with_return_values.md`: getting results back with `return`
- `lesson_5_default_parameters.md`: simple default values
- `lesson_6_real_life_function_programs.md`: practical function-based programs
- `module_quiz.md`: quiz with answer key
- `student_practice_sheet.md`: printable practice sheet
- `code_examples/`: short runnable Python files for this module

Recommended use:

1. Emphasize reuse and organization, not abstraction for its own sake.
2. Show the difference between defining a function and calling a function.
3. Keep parameter examples small and familiar.
4. Introduce `return` only after students are comfortable with function calls.
