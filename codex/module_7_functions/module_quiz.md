# Module 7 Quiz: Functions

## Part A: Multiple Choice

1. What is a function?
   - A. a repeated error
   - B. a reusable block of code
   - C. a data type
   - Answer: B

2. What keyword is used to define a function?
   - A. `for`
   - B. `return`
   - C. `def`
   - Answer: C

3. What is a parameter?
   - A. a loop counter
   - B. an input name in a function definition
   - C. an error message
   - Answer: B

4. What does `return` do?
   - A. repeats a loop
   - B. sends a value back from a function
   - C. prints a string
   - Answer: B

5. When is a default parameter used?
   - A. when no new value is passed
   - B. always
   - C. never
   - Answer: A

## Part B: Short Answer

1. What is the difference between defining and calling a function?
2. What is the difference between a parameter and an argument?
3. What is the difference between `print()` and `return`?
4. Why are functions useful?

## Part C: Coding

1. Write a function that prints `Hello`.
2. Write a function that takes a name parameter and prints `Welcome ___`.
3. Write a function that returns the sum of two numbers.

## Answer Key

### Part B Suggested Answers

1. Defining creates the function, while calling runs it.
2. A parameter is the name in the function definition, while an argument is the value passed during the call.
3. `print()` shows output, while `return` gives a value back to the program.
4. Functions reduce repetition and make programs easier to organize.

### Part C Suggested Solutions

```python
def say_hello():
    print("Hello")
```

```python
def greet(name):
    print(f"Welcome {name}")
```

```python
def add_two_numbers(a, b):
    return a + b
```
