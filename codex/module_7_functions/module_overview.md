# Module 7 Overview: Functions

## Module title

Functions

## Recommended level

Beginner students who have completed Module 6

## Duration

6 lessons, 40 to 45 minutes each

## Module purpose

This module teaches students how to group code into reusable pieces called functions. Students learn how to define a function, call it, send information into it using parameters, and get information back using `return`.

## Learning objectives

By the end of this module, students will be able to:

- explain what a function is
- define and call a simple function
- use parameters to pass values into a function
- use `return` to send a result back
- use simple default parameters
- write short programs that use multiple functions

## Prior knowledge

Students should be able to:

- use variables and input
- write `if` statements
- use loops
- read indented code blocks

## Module progression

### Lesson 1

- Focus: what a function is
- Outcome: students understand the idea of reusable code

### Lesson 2

- Focus: defining and calling simple functions
- Outcome: students can create and run a basic function

### Lesson 3

- Focus: parameters
- Outcome: students can pass values into functions

### Lesson 4

- Focus: return values
- Outcome: students can get results back from functions

### Lesson 5

- Focus: default parameters
- Outcome: students can use simple optional values

### Lesson 6

- Focus: real-life function programs
- Outcome: students can organize a program into functions

## Teacher reminders

- Compare functions to reusable recipes or named tasks.
- Show clearly that defining a function does not run it automatically.
- Use small functions before multi-step ones.
- Use board tracing for parameter values and returned results.
- Keep naming clean and meaningful.

## Common misconceptions

- Students may think writing `def` runs the function immediately.
- Students may confuse a parameter with a normal variable.
- Students may print inside a function when they actually need `return`.
- Students may forget brackets when calling a function.

## End-of-module evidence

A student is ready for Module 8 if they can:

- define and call their own function
- use one or two parameters correctly
- return a simple value
- explain the difference between printing and returning
- organize a short program into smaller functions
