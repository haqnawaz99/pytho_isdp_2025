# Student Practice Sheet: Module 7

## Instructions

Read each task carefully. Write the function, then call it if needed.

## Section 1: Simple Functions

1. Write a function that prints your city.
2. Write a function that prints a study reminder.
3. Write a function that prints a greeting.

## Section 2: Calling Functions

1. Define a function and call it once.
2. Define a function and call it three times.
3. Define two functions and call both.

## Section 3: Parameters

1. Write a function that takes a name and prints `Hello ___`.
2. Write a function that takes a city and prints `I live in ___`.
3. Write a function that takes subject and marks and prints both.

## Section 4: Return Values

1. Write a function that returns `25`.
2. Write a function that returns the sum of 2 numbers.
3. Write a function that returns a full greeting sentence.

## Section 5: Default Parameters

1. Write a function with default name `Student`.
2. Write a function with default city `Lahore`.
3. Write a function with default subject `Python`.

## Section 6: Real-Life Functions

1. Make a total cost function.
2. Make a marks message function.
3. Make a weather advice function.

## Challenge Section

1. Write a small program with two functions working together.
2. Use both a parameter and a return value in one function.
3. Explain the difference between printing a value and returning a value.

## Teacher checking guide

Check whether students can:

- define and call functions correctly
- pass values into parameters
- use `return` when needed
- use simple default parameters
- split a task into smaller functions
