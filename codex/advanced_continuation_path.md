# Advanced Continuation Path

## Purpose

This file maps what should come after the beginner course.

## Recommended next-phase modules

### Module A: Setup and Coding Habits

- comments
- file organization
- running files confidently

### Module B: Error Handling

- common runtime errors
- `try` and `except`
- user-friendly input handling

### Module C: File Input and Output

- reading files
- writing files
- simple saved records

### Module D: Larger Projects

- report card system
- attendance tracker
- library or maktaba tracker
- quiz game

## Why this is needed

The beginner course currently ends at the right place for first-time learners. These extension topics should be taught only after students are secure with Modules 1 to 9.
