# Setup Guide — Python & VS Code
### Step-by-step for Students (Ages 10–16)

---

## Step 1: Install Python

1. Open your browser and go to: **python.org/downloads**
2. Click the big yellow button **"Download Python 3.x.x"**
3. Run the downloaded file
4. **IMPORTANT:** On the first screen, tick the box that says **"Add Python to PATH"**
5. Click **"Install Now"**
6. Wait for it to finish, then click **Close**

**To check Python is installed:**
- Open the Start menu, search for **"Command Prompt"** or **"PowerShell"**
- Type: `python --version`
- You should see something like: `Python 3.12.0`

---

## Step 2: Install VS Code

1. Go to: **code.visualstudio.com**
2. Click **"Download for Windows"**
3. Run the installer — accept all defaults
4. When asked about "Additional Tasks", tick **"Add to PATH"** and **"Open with Code"**

---

## Step 3: Set Up VS Code for Python

1. Open VS Code
2. Click the **Extensions** icon on the left sidebar (looks like 4 squares)
3. Search for **"Python"** (by Microsoft)
4. Click **Install**

---

## Step 4: Run Your First Program

1. Open VS Code
2. Go to **File → Open Folder** and open your course folder
3. Click on any `.py` file to open it
4. Press **F5** or the **▶ Run** button at the top right
5. You should see output in the **Terminal** at the bottom

**Shortcut to run:** Press `Ctrl + F5`

---

## Step 5: Create a New Python File

1. In VS Code, go to **File → New File**
2. Save it as something like `my_practice.py` (the `.py` is important)
3. Type your code and press `Ctrl + F5` to run

---

## Common Problems

| Problem | Solution |
|---------|----------|
| `python` not recognized | Re-install Python and tick "Add to PATH" |
| VS Code shows red underlines | Install the Python extension (Step 3) |
| Code runs but nothing shows | Make sure you have a `print()` statement |
| IndentationError | Check your spaces — Python needs consistent indentation |

---

## Folder Structure for This Course

Save all your work in one folder, for example:
```
C:\Users\YourName\Documents\Python\
```

Download or copy the course files into this folder.
