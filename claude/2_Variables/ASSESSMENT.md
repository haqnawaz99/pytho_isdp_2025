# Module 2 — Assessment

## Pass Criteria
Student can independently:
- [ ] Create a variable with a valid name
- [ ] Store user input in a variable
- [ ] Use .upper(), .lower(), .title() on a string variable
- [ ] Explain what a variable is in their own words

## Quick Check Questions (ask orally)
1. Can a variable name start with a number? (Expected: No)
2. What does .title() do to "ahmed khan"? (Expected: "Ahmed Khan")
3. What does len(name) do if name = "Ali"? (Expected: returns 3)

## Quiz Marking (2_8_quiz.py)
- Easy: 5 marks
- Medium: 6 marks
- Complex: 6 marks
- **Total: 17 marks**

## Grade Scale
| Marks | Grade |
|-------|-------|
| 15–17 | Excellent |

| 7–10  | Needs revision |

## Practical Submission
Students write a `my_profile.py` that:
1. Asks for name (apply .title())
2. Asks for city (apply .upper())
3. Asks for favourite subject
4. Prints a formatted profile using all three variables
5. Prints the length of their name
