# Module 2 — Variables: Teacher Notes

## Duration
2 classes (45–60 minutes each)

## Learning Objectives
By end of module, students can:
1. Create a variable and assign a value
2. Use variables in print() and f-strings
3. Follow Python naming rules
4. Use string methods: upper(), lower(), title(), strip()

## Key Concept to Emphasize
A variable is a labelled box — whatever is in the box changes when you reassign.
Show: `name = "Ahmed"` then `name = "Fatima"` — the box now holds a new value.

## Common Mistakes to Watch
- `2name = "Ahmed"` — cannot start with a number
- `my name = "Ahmed"` — no spaces in variable names
- Forgetting `.strip()` when input has accidental spaces
- Case matters: `Name` and `name` are different variables

## Tips for Urdu Delivery
- "Variable matlab dabba jisme value rakhte hain"
- "Naam dete hain dabbe ko, jaise `age = 15`"
- `.upper()` — "Sab capital ho jaata hai"
- `.title()` — "Har lafz ka pehla harf bada ho jaata hai"
