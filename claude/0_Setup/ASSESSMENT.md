# Module 0 — Assessment

## Pass Criteria
Student can independently:
- [ ] Run a Python file in VS Code
- [ ] Write a print() statement with their name
- [ ] Add at least one comment to code
- [ ] Identify the line number from an error message

## Quick Check Questions (ask orally)
1. What does print() do? (Expected: shows text on screen)
2. How do you write a comment in Python? (Expected: start with #)
3. If you see "SyntaxError on line 3", what does that tell you? (Expected: mistake on line 3)

## Quiz Marking (0_5_quiz.py)
- Easy (Q1–Q5): 1 mark each = 5 marks
- Medium (Q6–Q8): 2 marks each = 6 marks
- Complex (Q9–Q10): 3 marks each = 6 marks
- **Total: 17 marks**

## Grade Scale
| Marks | Grade |
|-------|-------|
| 15–17 | Excellent — Ma sha Allah |
| 11–14 | Good — keep practising |
| 7–10  | Needs revision |
| 0–6   | Revise module before moving on |

## Practical Submission
Ask students to create a file `about_me.py` that:
1. Prints their name
2. Prints their city
3. Prints their favourite subject
4. Has at least 2 comments
