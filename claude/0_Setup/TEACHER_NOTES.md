# Module 0 — Setup: Teacher Notes

## Duration
1 class (45–60 minutes)

## Learning Objectives
By end of class, students can:
1. Open VS Code and run a Python file
2. Write and run a print() statement
3. Add comments to code
4. Read a basic error message and find the line number

## Lesson Flow

| Time | Activity |
|------|----------|
| 0–10 min | Install Python + VS Code together (or pre-installed) |
| 10–20 min | 0_1_what_is_python.py — What is Python? Why learn it? |
| 20–30 min | 0_2_your_first_program.py — First print(), run it live |
| 30–40 min | 0_3_comments.py — Comments with # |
| 40–50 min | 0_4_common_errors.py — Show errors, read the message |
| 50–60 min | Quiz + wrap up |

## Common Mistakes to Watch
- Students forget closing quote or bracket → SyntaxError
- Students type `Print()` with capital P → NameError
- Mixing single and double quotes on same string
- Not saving the file before running (Ctrl+S first!)

## Tips for Urdu Delivery
- Say: "print() matlab computer ko bolna — yeh line screen pe dikhao"
- Show errors deliberately — "galti se darna nahin, error message helper hai"
- Let students type the first program themselves — muscle memory matters
