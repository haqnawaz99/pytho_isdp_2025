## Folder Structure

Each module folder contains:

```
N_ModuleName/
├── N_1_lesson.py          ← lesson code (run in class, explain line by line)
├── N_2_lesson.py          ← next lesson
├── ...
├── N_X_applications.py    ← real-world examples combining concepts
├── N_Y_quiz.py            ← student practice (Easy / Medium / Complex)
├── N_Z_quiz_solution.py   ← solutions (teacher reference)
├── TEACHER_NOTES.md       ← duration, objectives, tips, common mistakes
├── ASSESSMENT.md          ← rubric, quiz marking guide, pass criteria
└── practice_sheet.md      ← printable offline exercises
```

Module 10 (Projects) also contains:
- `10_X_*_starter.py` — give this to students
- `10_X_*_solution.py` — full working solution

## How to Use This Course

### For the Teacher
1. Read `TEACHER_NOTES.md` before each class
2. Open the lesson `.py` files on the projector / screen share
3. Run code live — let students predict output before running
4. Hand out `practice_sheet.md` for offline practice (can be printed)
5. Use `ASSESSMENT.md` for marking quiz submissions

### For the Student
1. Watch the YouTube video for the module
2. Open the lesson `.py` files and run them
3. Complete the quiz file (Easy first, then Medium, then Complex)
4. Check your answers in the solution file

---

## Recommended Teaching Order

Follow modules 0 → 10 in sequence. Do not skip Module 8 (Error Handling) — it is needed for Module 9 and the projects.

## Prerequisites
- Python 3.x installed (see `SETUP.md`)
- VS Code or any text editor
- No prior coding experience needed
