# Module 1 — Assessment

## Pass Criteria
Student can independently:
- [ ] Concatenate two strings using +
- [ ] Use input() to get a name from the user
- [ ] Store input in a variable and print it
- [ ] Use len() on a string

## Quick Check Questions (ask orally)
1. How do you join two strings in Python? (Expected: using +)
2. What does input() do? (Expected: asks the user to type something)
3. What does len("Ahmed") return? (Expected: 5)

## Quiz Marking (1_4_quiz.py)
- Easy (Q1–Q5): 1 mark each = 5 marks
- Medium (Q6–Q8): 2 marks each = 6 marks
- Complex (Q9–Q10): 3 marks each = 6 marks
- **Total: 17 marks**

## Grade Scale
| Marks | Grade |
|-------|-------|

## Practical Submission
Students write a program that:
1. Asks user for first name and last name separately
2. Joins them and prints: "Assalamu Alaikum, [Full Name]!"
3. Prints the total length of the full name
