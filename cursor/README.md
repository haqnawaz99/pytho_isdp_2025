# Python Course (Redesigned) — ISDP 2025

This folder contains the **enhanced course** for mixed **madrasa and school** students in Pakistan. All redesign work lives here only; original modules in `code/1_Strings`, `code/2_Variables`, etc. stay unchanged for reference.

## Teaching approach

| Item | Policy |
|------|--------|
| **Code** | English only (Python keywords, variable names, strings in examples) |
| **Classroom language** | Mix Urdu and English (teacher explains in both) |
| **Examples** | Islamic context + Pakistani daily life (school, bazaar, cricket, cities) |
| **Tools** | Laptop + **Visual Studio Code** + Python 3.10+ |
| **Pace** | Flexible — cover all modules; time per lesson may vary |

## Folder structure

```
cursor/
  README.md                 ← You are here
  COURSE_MAP.md             ← Full module sequence
  SETUP_VS_CODE.md          ← Install Python + VS Code (labs)
  0_Getting_Started/
  1_Strings/
  2_Variables/              ← (next phase)
  ...
```

Each module includes:

- `CHAPTER_*.md` — Student-facing chapter (English) with Urdu vocabulary boxes
- `TEACHER_NOTES.md` — Delivery tips, misconceptions, extensions
- `ASSESSMENT.md` — Quizzes, checkpoints, rubric
- `*.py` — Runnable lesson and practice files (no emoji in source)

## How to run a lesson

1. Open this folder in VS Code: **File → Open Folder** → select `cursor`.
2. Open the lesson `.py` file (e.g. `1_Strings/1_1_concatenation.py`).
3. Press **Run Python File** (play button) or right-click → **Run Python**.
4. Teach concept from `CHAPTER_*.md` first, then run code together.
