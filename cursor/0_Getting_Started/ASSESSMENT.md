# Module 0 — Assessment

## Formative checkpoint (end of module)

**Practical:** Student submits `my_intro.py` that:

- [ ] Has at least 2 comments
- [ ] Prints at least 4 lines (name, city, school/madrasa, hobby)
- [ ] Runs without error in VS Code

**Oral (30 seconds):** What does `print()` do? What does `#` do?

---

## Quiz — Multiple choice

1. A Python program is:
   - a) A set of instructions for the computer  
   - b) A Microsoft Word file  
   - c) Only for games  

2. We write Python code mostly in:
   - a) Urdu script only  
   - b) English keywords and symbols  
   - c) Arabic only  

3. `print("Hello")` shows output because:
   - a) `print` displays text  
   - b) `Hello` is a comment  
   - c) Python saves it to USB  

4. Which line is a valid comment?
   - a) `// hello`  
   - b) `# hello`  
   - c) `-- hello`  

5. `Print("hi")` fails because:
   - a) Python needs capital Print  
   - b) Python is case-sensitive — use `print`  
   - c) hi is too short  

**Answers:** 1-a, 2-b, 3-a, 4-b, 5-b
