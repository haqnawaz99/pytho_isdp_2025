## Lab tips (VS Code)

1. Run file → Terminal must be **focused** for typing at `input` prompt.
2. If program “hangs”, it is usually **waiting for input** — type and press Enter.
3. Pair work: one reads prompt aloud in Urdu, one types.

## Common misconceptions

| Issue | Response |
|-------|----------|
| `input()` returns number | Always string until Module 3 |
| `name = Ahmed` | Strings need quotes |
| `print(age + " years")` with int | Use comma or `str()` — preview only in 2.3 |
| Copy-paste wrong variable name | Run line by line; teach careful spelling |
