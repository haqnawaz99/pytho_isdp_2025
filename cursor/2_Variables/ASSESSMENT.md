## Multiple choice

1. `name = "Sara"` means:
   - a) Sara is a function  
   - b) The variable name stores the value `"Sara"`  
   - c) name and "Sara" are the same thing  

2. `input()` always returns:
   - a) int  
   - b) str (text)  
   - c) float  

3. Which variable name is **valid**?
   - a) `student name`  
   - b) `student_name`  
   - c) `1student`  

4. `print("Length:", len("Lahore"))` works because:
   - a) comma allows number after text  
   - b) len returns a string  
   - c) Lahore is an int  

5. `Name` and `name` in Python are:
   - a) The same variable  
   - b) Different variables (case-sensitive)  
   - c) Both invalid  

**Answers:** 1-b, 2-b, 3-b, 4-a, 5-b
