## Multiple choice

1. `def greet():` needs:
   - a) colon and indented body  
   - b) semicolon at end  
   - c) return always  

2. `greet("Ali")` — `"Ali"` is the:
   - a) argument  
   - b) return  
   - c) default  

3. `return x` means:
   - a) print x only  
   - b) send x back to caller  
   - c) delete x  

4. `def f(name="Guest"):` — calling `f()` uses:
   - a) Guest  
   - b) error  
   - c) empty string only  

5. Functions help us:
   - a) avoid repeating code  
   - b) slow the program  
   - c) remove variables  

**Answers:** 1-a, 2-a, 3-b, 4-a, 5-a

---

## Short answer

1. Difference between parameter and argument?  
2. When use `return` instead of `print` inside a function?

**Sample:**  
1. Parameter in def; argument is value when calling.  
2. When caller needs the value for later use or maths.
