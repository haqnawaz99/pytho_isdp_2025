## Do NOT re-teach Module 4

If students forget `>=` or `and`, point to Module 4 handout — spend time on **colon and indent** instead.

## Indentation (most common failure)

1. Show error when indent missing (live demo).
2. VS Code: select block → Tab to indent, Shift+Tab to unindent.
3. Rule: **colon + new line + 4 spaces**.

Draw on board:

```
if marks >= 50:
    print("Pass")    ← belongs to if
print("Done")        ← always runs
```

## Sensitive topics

- Prayer / Quran yes-no: encourage, never shame.
- Fee paid: use hypothetical office scenario.
- Gender greetings: optional; many classes skip Example in old files.

## while loop safety

Before running, ask: “What variable changes? Will it eventually stop?”  
If laptop hangs on infinite loop: **Stop** (square button in VS Code toolbar).

## for / range

Write on board: `range(1, 6)` → 1,2,3,4,5 (6 excluded).  
Connect to Module 7 lists — “we will do more with lists soon.”
