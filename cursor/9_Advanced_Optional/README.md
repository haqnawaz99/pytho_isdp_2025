## Prerequisites

Students should complete **cursor Modules 0–8** first.

## Why separate?

- `try/except` and file paths are hard on Windows labs.  
- Report card capstone in claude uses `open()` — too much for first exposure.  
- Keeps Year 1 focused on thinking in Python, not filesystem issues.
