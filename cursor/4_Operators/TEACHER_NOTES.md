## Critical teaching points

### `=` vs `==`

Write on board:

- `age = 12` → store  
- `age == 12` → question (True/False)

### `//` and `%` together

Do one story on board: 28 items, 5 groups. Draw 5 groups of 5 + 3 left. Link to `//` and `%`.

### Division surprise

`12 / 4` is `3.0` not `3` — mention float; do not dwell until needed.

### Logical operators before `if`

Say: “Today we **print** True/False. Next module we **act** on it with `if`.”

## Common errors

| Error | Fix |
|-------|-----|
| `if marks = 70` | Use `==` in Module 5; now only `marks == 70` in expression |
| `true` lowercase | Must be `True` |
| `10 + 5 * 3` = 45 | BODMAS demo on board |
| f-string forgets `f` | `f"Hello {name}"` |
