# Project Showcase and Reflection (Module 8)

Adopted from **codex** `lesson_6_project_showcase_and_reflection.md`.

**Duration:** 1–2 periods (flexible)

## Learning objective

Students present their project and reflect on what they learned in the Python course.

## Showcase format (2 minutes per student)

1. **Name and project title** (10 sec)  
2. **Demo** — run program once with sample input (60 sec)  
3. **One Python concept** — point to `if`, loop, function, or list (30 sec)  
4. **One challenge** — what was hard and how you fixed it (20 sec)  

## Audience rules

- Listen quietly; clap after each demo.  
- One question per presenter (optional).  

## Written reflection (hand in)

Answer in Urdu or English (5–8 sentences total):

1. What was your favourite project and why?  
2. What is one thing Python can do that you could not do before?  
3. What do you want to learn next in programming?  
4. How did you help a classmate, or how did someone help you?
