# Module 8 — Teacher Notes (Mini Projects)

## Time guide (flexible)

| Project | Time | Starter | Solution |
|---------|------|---------|----------|
| 1 Attendance | 1–2 periods | `8_1_attendance_starter.py` | `8_1_attendance_solution.py` |
| 2 Maktaba | 1–2 periods | `8_2_maktaba_starter.py` | `8_2_maktaba_solution.py` |
| 3 Fee | 2 periods | `8_3_fee_calculator_starter.py` | `8_3_fee_calculator_solution.py` |
| 4 Name formatter | 1 period | `8_4_name_formatter_starter.py` | `8_4_name_formatter_solution.py` |
| 5 Class report | 2–3 periods | `8_5_class_report_starter.py` | `8_5_class_report_solution.py` |

Total: roughly **2–3 weeks** if 2–3 class periods per week.

## Teaching strategy

1. **Do not** show solution on day 1.  
2. Live-code the first TODO on projector from starter file.  
3. Students finish remaining TODOs.  
4. Peer review: partner runs partner's program.  
5. Optional demo day: 2-minute presentation per student.

## Differentiation

| Level | Task |
|-------|------|
| Struggling | Complete projects 1–2 only; use solution with tracing |
| Core | Projects 1–4 |
| Advanced | Project 5 + add `input()` to add a student to list |

## Pairing mixed madrasa + school

- Same starter code; students customize strings (school name, subjects, books).
- Avoid grading whose cultural example is “better.”

## Common issues

| Issue | Fix |
|-------|-----|
| Negative present if absent > total | Add `if absent > total: print error` (extension) |
| Empty list in project 2 | Start with 3 books in starter data |
| `student['name']` KeyError | Check spelling of keys in dict |

## Course finale

- Issue simple **certificate checklist** from `ASSESSMENT.md`.
- Point to next steps: Python files, turtle graphics, or continue next year.

## Do not assign as homework without lab

Projects need `input()` — hard to grade on paper only.
