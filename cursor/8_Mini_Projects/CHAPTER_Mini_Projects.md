# Module 8: Mini Projects (Course Capstone)

**Prerequisites:** Modules 0–7  
**Goal:** Combine everything in small, real programs students can explain in class.

---

## Module learning objectives

Students will be able to:

1. Plan a program in steps (input → process → output).
2. Complete **four guided projects** with starter files.
3. Build one **integrated class report** using lists, dictionaries, `if`, loops, and functions.
4. Present their program: what it does, one challenge they solved.

> **Urdu:** منی پروجیکٹ = سب ماڈیولز کو ایک چھوٹے پروگرام میں جوڑنا۔ پہلے سوچیں، پھر کوڈ لکھیں۔

---

## How to work on a project

1. Read the **project brief** below.  
2. Open the **`_starter.py`** file — fill in `# TODO` parts.  
3. Run often in VS Code after each small step.  
4. Compare with **`_solution.py`** only after trying (or for teacher demo).  
5. Change examples to your own school/madrasa/city.

---

## Project 1 — Class attendance counter

**Skills:** variables, `input`, subtraction, `if`/`else`, f-strings  

**Brief:** Ask total students and absent students. Print how many are present. If absent is 0, print a special message.

**Files:** `8_1_attendance_starter.py`, `8_1_attendance_solution.py`

---

## Project 2 — Maktaba book list

**Skills:** lists, `append`, `for` loop, f-strings  

**Brief:** Keep a list of book titles. Print numbered list. Add one book from `input`. Print updated list.

**Files:** `8_2_maktaba_starter.py`, `8_2_maktaba_solution.py`

---

## Project 3 — School fee calculator

**Skills:** `int`/`float` input, operators, functions (optional), f-strings  

**Brief:** Ask monthly fee and number of months (or students). Print total. Apply simple discount if total above a limit.

**Files:** `8_3_fee_calculator_starter.py`, `8_3_fee_calculator_solution.py`

---

## Project 4 — Student name formatter

**Skills:** strings, `.strip()`, `.title()`, variables, f-strings  

**Brief:** Ask for full name (maybe messy spaces). Print clean title-case name and length.

**Files:** `8_4_name_formatter_starter.py`, `8_4_name_formatter_solution.py`

---

## Project 5 — Class report (integrated capstone)

**Skills:** list of dicts, `for`, `if`, functions, f-strings  

**Brief:** Store 3+ students with name, marks, city. Print report, count pass/fail, average marks (simple loop sum).

**Files:** `8_5_class_report_starter.py`, `8_5_class_report_solution.py`

---
