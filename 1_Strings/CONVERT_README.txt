================================================================================
Converting the Strings Chapter to DOCX and PDF
================================================================================

DOCX (Word)
-----------
A Word document with the full chapter and embedded diagrams is generated by:

   python md_to_docx.py

Output: CHAPTER_Strings.docx (in this folder)

Requirements: pip install python-docx

The script reads CHAPTER_Strings.md and inserts images from the diagrams/
folder (string_sequence.png, concatenation.png, immutability.png,
string_methods_overview.png).


PDF
---
To get a PDF:

  Option 1 (recommended): Open CHAPTER_Strings.docx in Microsoft Word and use
  File > Save As > PDF.

  Option 2: Use an online converter (e.g. Word to PDF) or LibreOffice:
  Open the DOCX in LibreOffice Writer and use File > Export As > Export as PDF.


Diagrams
--------
The diagrams folder contains:

  string_sequence.png   - String as a sequence of character boxes
  concatenation.png     - Joining strings with +
  immutability.png     - Original string unchanged; methods return new string
  string_methods_overview.png - Overview of .upper(), .lower(), .strip(), etc.

These are referenced in CHAPTER_Strings.md. When you convert to DOCX using
md_to_docx.py, they are embedded automatically. To add more diagrams, place
PNG/JPG files in diagrams/ and add a line in the Markdown:

  ![Description](diagrams/your_image.png)

Then run md_to_docx.py again.
